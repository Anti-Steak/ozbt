ozbt core command documentation

Help

Parameters within < greater than and less than > symbols are required. Parameters within [ block brackets | separated with pipes ] are use one of.

User levels

There are six (6) user levels currently, used for permissions. Anything above is included in the below items, so a moderator can do anything a regular can do but cannot do something a supermoderator can do. When using these with commands, use the word but in lowercase: Everybody -> everybody or SuperModerator -> supermoderator.

  1. Everybody
  2. Regular
  3. Subscriber
  4. Moderator
  5. SuperModerator
  6. Broadcaster

!access Set permission for a custom command.

!access <custom command> <userlevel>

megadriving: !access !nextstream moderator
ozbt: megadriving -> Access for "nextstream" set to "moderator".

!cmd Create, update or delete a custom command.

By default, all commands are open for everybody to use, follow up this command with !access if you wish to change it.

!cmd [add|edit|delete] <commandName> <output text>

megadriving: !cmd add !nextstream Next stream is in two days.
ozbt: megadriving -> command "!nextstream" was created.
megadriving: !cmd edit !nextstream I changed my mind, it's in a week.
ozbt: megadriving -> command "!nextstream" was updated.
megadriving: !cmd delete !nextstream
ozbt: megadriving -> command "!nextstream" was deleted.

!greet Create, update or delete text that greets a user if they subscribe or resubscribe.

Greetings have a few variables you can use, listed below.

  1. $username -> The users username.
  2. $months$ -> (resub only) The number of months subscribed.
  3. $s$ -> (resub only) if you wish to have words like "4 months resub" instead of "4 month resub". Will only appear if the months is more than 1.

!greet [sub|resub] <output text>
!greet delete [sub|resub]

megadriving: !greet sub Thanks for subscribing $username$~
ozbt: megadriving: Greeting for "sub" has been updated.
megadriving: !greet resub Thanks for resubscribing $username$~ That's $months$$s$ now!
ozbt: megadriving: Greeting for "resub" has been updated.
megadriving: !greet delete sub
ozbt: Greeting for "sub" has been deleted.

!regular Add or remove a regular.

Depending on your stream, they can have the same, different or special commands. It's up to you! Note: Usernames are not checked. Be aware of this.

!regular [add|delete] <username>

megadriving: !regular add tirean
ozbt: Regular "tirean" was added.

megadriving: !regular add tirean
ozbt: Regular "tirean" already exists.

megadriving: !regular delete tirean
ozbt: Regular "tirean" was deleted.

!spoll Create or get results from a strawpoll

This a slightly more complicated command, you need to use parameters preceded by flags. You can only use one -t flag to set the title but as many -o flags to set options as you like. Bear in mind your option includes a -o, it will separate them into separate options.

!spoll -t Title of poll -o Option 1 -o Option 2 -o Option 3 !spoll results

megadriving: !spoll -t Is ozbt a cool Twitch bot? -o Absolutely! Love it! -o It's alright.. -o It's awful pls stop
ozbt: Vote on "Is ozbt a cool Twitch bot?" here: http://strawpoll.me/7255934
megadriving: !spoll results
ozbt: megadriving -> Results for "Is ozbt a cool Twitch bot?": Absolutely! Love it!: 100 It's alright..: 101 It's awful pls stop: 999

!subgoal Create, edit, update or delete subscriber goals

A "subgoal" is a goal that increases when someone subscribes. It could be useful for an incentive to subscribe as a vote to learn how to speedrun a certain game or to play a certain game. The hash that is referenced is used to make modifications to the subgoals later.

!subgoal list
!subgoal [add|edit|delete] <subs> <Name of subgoal>
!subgoal modsubs [+|-]<points> (Super moderators only.)
!subgoal countresubs [yes|no]

megadriving: !subgoal add 10 Play Super Mario Bros.
ozbt: megadriving -> subgoal "Play Super Mario Bros." was created. (hash: "sMb")

megadriving: !subgoal list
ozbt: 1. Super Mario Bros. 0/10 (0%) (hash: sMb)

megadriving: !subgoal edit sMb 50 Super Mario Bros.
ozbt: megadriving -> subgoal "Play Super Mario Bros." was edited.

megadriving: !subgoal countresubs sMb yes
ozbt: megadriving -> Resubs will now count for the subgoal with hash "sMb".

megadriving: !subgoal modsubs sMb +10
ozbt: megadriving -> "Play Super Mario Bros." has been updated, now has 10.

megadriving: !subgoal list
ozbt: 1. Super Mario Bros. 5/50 (10%) (hash: sMb)

megadriving: !subgoal delete sMb
ozbt: megadriving -> subgoal "Play Super Mario Bros." was deleted.

!supermod Add or remove a super moderator.

Depending on your stream, they can have the same, different or special commands. It's up to you! Note: Usernames are not checked. Be aware of this.

!supermod [add|delete] <username>

megadriving: !supermod add tirean
ozbt: Regular "tirean" was added.
megadriving: !supermod add tirean
ozbt: Regular "tirean" already exists.
megadriving: !supermod delete tirean
ozbt: Regular "tirean" was deleted.

!uptime Shows the uptime of the stream.

(Not using night's code.)

!uptime

megadriving: !uptime
ozbt: megadriving -> Stream is offline.
megadriving: !uptime
ozbt: megadriving -> Streaming for 5:19.
